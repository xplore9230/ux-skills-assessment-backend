Visual style:

Solid dark background, no gradients (#050814 and #0C101C).

Minimal accent palette using soft gold (#F4B731) and muted orange (#F97316) or teal (#2DD4BF) for key highlights only.

Typography: Use an elegant serif like Newsreader or Playfair Display for main headings and a clean sans like Inter or Manrope for body and UI text. Headings should feel editorial and premium; body text should be highly legible.

Big breathing space, generous padding, soft rounded corners, subtle shadows.

Subtle background accent: a soft blurred glow behind the report card using the accent color.

Quiz screen:

Centered card with the question text, a small progress indicator ("Question 4 of 15" + horizontal progress bar), and five full-width pill-style options stacked vertically.

Clear selected state: solid accent background with white text.

Bottom sticky bar with "Previous" (ghost button) and "Next" (solid accent button).
